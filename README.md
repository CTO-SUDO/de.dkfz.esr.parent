# CTO SUDO Parent

A Maven parent POM artifact for all CTO SUDO Java 8 projects at the Clinical Trial Office (CTO) of the German Cancer Research Center (DKFZ) in Heidelberg.

## Overview

This project serves as the root Maven parent POM that provides centralized dependency management and build configuration for all CTO SUDO Java 8 projects. It ensures consistent versions across all child projects and simplifies maintenance.

## Build

```bash
mvn clean install
```
